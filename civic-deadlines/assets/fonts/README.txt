Place your licensed webfont files here so the app can embed them for every visitor.

Expected filenames:
- DegularDemo-Regular.otf
- DegularDemo-Bold.otf

Why this matters:
- If the font is only installed on your machine, other people will not see it.
- If you self-host the font files in this folder, the browser downloads them with the site.

Notes:
- Use webfont files you are licensed to serve on the web.
- If you choose a different font later, keep the same filenames or update the `@font-face` rules in `index.html`.
